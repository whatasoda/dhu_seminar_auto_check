このページでやること

  1. 別タブで、[デジキャン > 学内システム > 履修・成績・出席 > Web履修申請 > 履修チェック結果（時間割） > 成績状況参照]を開く。
  2. ページを全選択([ctrl/cmd] + A)、コピー([ctrl/cmd] + C)する。
  3. この下のテキストボックスにペーストし、変換ボタンを押す。

ゼミ申請フォームでやること

ゼミ申請フォームで次の操作をしてください。（GoogleChrome推奨）

  1. デベロッパーツールを開く（開き方は下のうちのどれか）
    • ページのなにもないところを右クリック → 検証
    • F12
    • Ctrl + Shift + J / Option + Command + J
  2. Consoleというタブを見つけて開く
  3. 「>」の右にテキストカーソルが出ている状態で、下のボックスの中身をコピペしてEnter
